# MyiOS_Demo

iOS UI / 交互样例集合。

## Demos
- Icon Showcase: 使用 [LiquidGlassKit](https://github.com/DnV1eX/LiquidGlassKit)（SPM 依赖）展示 Icon Effect、Container Merge、Slider、Switch、Lens。
- Card Swipe: 侧滑卡片交互示例。
- Legacy Clock: 旧版翻页时钟保留示例。
- Gelatin Button: 原生按钮示例，在 iOS 26 以下叠加 [Gelatin](https://github.com/Polarizz/Gelatin) 弹性效果。
- Liquid Glass: 基于 [Nikiteizzz/LiquidGlass](https://github.com/Nikiteizzz/LiquidGlass) 的可调参数示例，展示折射玻璃与边缘高光效果。
- Theseus Showcase: 展示 [valzevul/Theseus](https://github.com/valzevul/Theseus) 可实现的 Adaptive TabBar、Glass Slider/Switch、Glass View 与主题化能力示例。
